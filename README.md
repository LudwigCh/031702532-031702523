# 031702532-031702523

师生树<br>

一、代码相关<br>
------

##### 1、我们将css相关代码及图片数据保存在css文件夹下，将js相关代码保存在dist文件夹下。<br>
##### 2、html代码中的引用使用相对路径完成。<br>

二、操作相关
------

##### 1、进入主界面，将合法输入输入到文本框中，点击提交。<br>
##### 2、点击提交后跳转到下一界面，在此界面点击生成树按钮生成家族树。<br>
##### 3、得到生成树后双击节点或点击缩放按钮进行浏览。<br>

##### 4.生成关联树时需点击子树在父树中的相关节点，即可生成浏览。

##### 5、通过搜索框可以查找想要的节点。
